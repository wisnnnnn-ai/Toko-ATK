# Sistem Informasi Penjualan dan Persediaan Barang

Sistem Informasi Penjualan dan Persediaan Barang berbasis web yang dikembangkan menggunakan Laravel 11 dan MySQL untuk membantu pengelolaan data barang, stok, transaksi penjualan, laporan, dan manajemen akun pengguna.

## 📌 Fitur Utama

### Dashboard

* Ringkasan total barang
* Ringkasan stok barang
* Ringkasan transaksi
* Statistik sistem

### Data Barang

* Tambah barang
* Edit barang
* Hapus barang
* Detail barang
* Cetak data barang

### Kategori Barang

* Tambah kategori
* Edit kategori
* Hapus kategori
* Detail kategori

### Stok Barang

* Monitoring stok
* Restok barang
* Riwayat perubahan stok

### Transaksi Penjualan

* Keranjang belanja
* Perhitungan total otomatis
* Pengurangan stok otomatis
* Cetak nota transaksi
* Riwayat transaksi

### Laporan

* Laporan penjualan
* Laporan stok barang
* Cetak laporan

### Manajemen Akun

* Login
* Logout
* Kelola akun pengguna
* Hak akses Admin dan Karyawan

## 🚀 Instalasi

Clone repository:

```bash
git clone https://github.com/username/repository.git
```

Masuk ke folder project:

```bash
cd nama-project
```

Install dependency:

```bash
composer install
```

Copy file environment:

```bash
cp .env.example .env
```

Generate application key:

```bash
php artisan key:generate
```

Konfigurasi database pada file `.env`

Jalankan migrasi database:

```bash
php artisan migrate
```

Jalankan server:

```bash
php artisan serve
```

Akses aplikasi melalui:

```text
http://127.0.0.1:8000
```
